## Visao Geral

Tico e um companheiro de IA para desktop Windows que vive na tela do usuario — um cursor companheiro que te segue, ouve comandos de voz, ve a tela atraves de capturas de tela e responde em voz alta enquanto aponta para elementos na tela. Construido para interacao multimodal fluida, ele detecta automaticamente o idioma (PT-BR / EN), pesquisa na web quando necessario e roda nativamente no Windows com uma interface WPF customizada.

A proposta de valor principal: transformar o desktop em um espaco de trabalho conversacional — pergunte qualquer coisa ao Tico com as maos livres, e ele ve, ouve e responde em tempo real.

[Website](https://ticoai.app)

## Modulos da Plataforma

### Cursor Companheiro
Companheiro de desktop construido em WPF com fisica de molas, animacao de caminho Bezier, visualizacao de forma de onda e baloes de fala. Persistente na barra de tarefas com um icone personalizado do Tico na bandeja, tecla de atalho configuravel (padrao F8) e controles de pausar/retomar TTS — sem esconder a barra de tarefas do Windows.

### Voz e Fala
Loop de voz completo: captura de audio com gating RMS, transcricao via OpenAI Whisper (deteccao automatica de idioma) e respostas faladas via OpenAI TTS com pausar/retomar reproducao.

### Consciencia de Tela
Captura de tela em tempo real e analise de elementos na tela via OmniParser, alem de um sistema de apontamento (v1 + v2) que permite ao Tico destacar visualmente elementos da UI enquanto fala sobre eles.

### Motor de IA
Integracao com Anthropic Claude com busca web integrada (`web_search_20250305`, ate 3 buscas por requisicao), gerenciamento de historico de conversas e streaming de contexto multi-turno via SSE.

### Plataforma Web
Site de marketing em Next.js 16 com landing page, fluxo de autenticacao, dashboard, blog de atualizacoes (baseado em markdown com curtidas e comentarios), paginas legais e i18n em dois idiomas (~160 chaves em PT-BR e EN). SSR via Supabase com estilizacao Tailwind 4.

### Pagamentos e Cobrancas
Infraestrutura de pagamento dupla: PIX via AbacatePay (BRL) e cartao de credito via Stripe (USD). Tres planos (gratuito / pro / ilimitado) com rastreamento de uso, webhooks idempotentes e Stripe Customer Portal para gestao de assinaturas.

### Autenticacao e Identidade
Supabase Auth com verificacao de email, recuperacao de senha, magic links e autorizacao baseada em JWT no proxy. Templates de email HTML customizados entregues via Resend SMTP.

### Sistema de Atualizacao
Verificador de atualizacoes automatico na inicializacao, com artefatos de instalador e `version.json` servidos pelo Cloudflare R2.

## Stack Tecnologico

**Desktop**: C# .NET 8 (WPF), DPAPI para armazenamento de tokens
**Backend / Proxy**: Cloudflare Workers (prod), Node.js (dev)
**Frontend**: Next.js 16, React, Tailwind 4, TypeScript
**Auth e Banco de Dados**: Supabase (PostgreSQL, RLS, RPCs)
**IA**: Anthropic Claude API (+ Web Search), OpenAI (Whisper + TTS)
**Pagamentos**: AbacatePay (PIX / BRL), Stripe (Cartao / USD)
**Email**: Resend (SMTP customizado)
**Armazenamento / CDN**: Cloudflare R2, Cloudflare DNS
**Infraestrutura**: Vercel, Cloudflare Workers, Supabase Pro
**Testes**: xUnit, NSubstitute
